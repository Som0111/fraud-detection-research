# Put the dataset here

Download `creditcard.csv` from the Kaggle Credit Card Fraud Detection dataset
and place it in this folder:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Expected file: `data/creditcard.csv`
(~144 MB, 284,807 rows; columns: Time, V1..V28, Amount, Class)

If this file is absent, the notebooks fall back to a synthetic stand-in so the
pipeline still runs — see the banner printed in notebook 01.
